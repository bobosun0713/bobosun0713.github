1.以下是用.net webForm(VB) 寫的，在百度的文字識別創建應用，再去應用管理則可以看到 APP_ID、API_KEY、SECRET_KEY 等資料，再去 AI 平台下載 API 接口程序。 https://ai.baidu.com/sdk#ocr (如果是用 VB 寫就選擇 C#SDK)
