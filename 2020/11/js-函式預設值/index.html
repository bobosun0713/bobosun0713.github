JS-函式預設值

十一月 4, 2020 分類 JavaScript學習

函示預設值兩種寫法 ES5 / ES6

在ES5中我們函式的預設值寫法為 :

function SomePeole (name , age , place){
    this.name = name !== undefind ? name : '某人';
    this.age = age !== undefind ? age : 20;
    this.place = place !== undefind ? place : '地區';
}

在ES6 中我們函式的預設值可以這樣寫 :

function SomePeole (name='某人' , age=20 , place='地區'){

}

這兩段就可以看的出在 ES6 程式碼在簡化的過程中的差異性!

但這兩個預設值有限制，如果傳入的參數式null，函式就不會採用預設值，會判定是自己給NULL，這部分需要注意。

以下特性都不會使用預設值 :

  • 0 , null , '’ / 數字0 、 空值 、 空字串
