第一次接觸Vue 來寫個轉盤吧！

十二月 28, 2020 分類 Ｖue學習

剛接觸框架最快上手的方式，就是動手作！

用 Vue3 來寫轉盤吧！

首先在撰寫 Vue 版本的時候，因為工作上的需要，已經寫過ＪＳ的版本了，會想使用 Vue 再次寫一遍，除了一方面之前工作，公司前端部分沒用框架，幾乎都是ＪＳ或 jQuery 來做前端開發，也因為這樣所以想要增進自己前端的技術能力，在下班時間慢慢學習 Vue 的部分．

撰寫轉盤的流程大致為：

  • 先用ＣＳＳ繪製出轉盤內園的扇行
  • 再用程式去控制，從資料勞取的數量在計算出每個區塊的角度
  • 畫面做好，計算要選轉的角度． 通常這部分都會給他預設轉個 5 圈加上原始角度，讓畫面看起來有在轉動．

撰寫完的心得

在剛開始寫 Vue3 的時候，最重要的是寫法上的差異需去適應，但在撰寫的過程中，也慢慢了解到 Vue 的好用之處，如 v-model 讓我在動態新增獎品時，給我很方便的雙向綁定等等好用的語法，在寫這轉盤我是用 Vue-Cli 專案去寫，雖然我這個可以用一般網頁引入 Vue 直接撰寫，但目前看線上課程已學玩 cli 部分，想趁這機會動手做看看用 cli 專案去寫，順便複習 components / props / emit ，來看看我吸收了多少．

最後的最後！

其實在學習 Vue3 之前，其實就有找時間看 Vue2 的部分，這次我以我自己的感受來說，用 Vue3 寫起來感覺比 Vue2 來的好上手，可能因為邏輯處理的部份，通集中在 setup()裡面，可以讓我知道更好規劃我程式撰寫流程及方便我更好閱讀，但這是我自己的感想，畢竟我寫 Vue 也沒有很久，最後也希望自己能更加熟練 Vue 這個框架，因為真的很好用呀呀！！！！
